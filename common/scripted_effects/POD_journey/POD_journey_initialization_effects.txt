# This effect lets Journeys be displayed in the sidebar GUI window.
# (meaning there could be other "hidden" journeys, e.g. triggered via character interaction)
#
# POD_add_to_journey_list needs to be called with 6 arguments:
# TYPE     - the journey ID, necessary to make everything else work
# GUITYPE  - can be either journey or great_journey, used in the sidebar window
# MONTHLY  - which script value to use for planning speed, used by POD_journey_monthly_tick
# GOLD     - upfront gold cost (can be 0)
# PRESTIGE - upfront prestige cost (can be 0)
# PIETY    - upfront piety cost (can be 0)
#
# NOTE:
# When you edit one of the Journeys here, also increment current_journey_version in common\script_values\POD_journey_values.txt
